# MEANx5
This exam is for a Bucket List manager.

You can find the code on GIT here: https://github.com/markneumann/MEANx5

To user this, you need to run:
* npm install
* bower install
* nodemon server.js

Then point your browser at
http://localhost:8001/#/dashboard

or
